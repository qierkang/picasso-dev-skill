# 服务端快捷触发语义

这份规则用于把用户的简短说法快速映射到服务端任务动作。

| 用户说法 | 标准动作 |
|----------|----------|
| `新任务` / `创建任务` | 读取 `服务端任务输入模板.md`，引导补齐任务输入 |
| `校验任务` / `校验所有任务` | 扫描 `incoming/` 并执行门槛校验 |
| `开启任务 <名称>` | 校验通过后从 `incoming` 移入 `active`，准备进入 `openspec` 流程 |
| `当前进度` | 汇总 `active/` 与 `archived/` 中各任务的 `status.md` |
| `完成任务 <名称>` | 进入归档流程 |
| `撤销任务 <名称>` | 进入撤销流程 |
| `需求变更` / `变更需求` | 读取 `服务端需求变更流程.md`，输出影响分析 |
| `补菜单数据 <名称>` | 基于已存在 design / archive 产物补 `MENU_DATA.md` |
| `补字典数据 <名称>` | 基于已存在 design / archive 产物补 `DICT_DATA.md` |
| `复制产出物 <名称>` | 将 `openspec` 产物复制到任务归档目录 |
