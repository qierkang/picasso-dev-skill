# Picasso 服务端任务规则

这组文档吸收自旧版 `picasso-task`，用于补齐 Picasso 服务端任务推进时的项目专属规则。

按需读取：

- 创建服务端任务输入时：`服务端任务输入模板.md`
- 查看任务与源码目录布局时：`服务端任务路径布局.md`
- 校验任务是否可开启时：`服务端任务门槛校验.md`
- 设计服务端方案时：`服务端设计约束.md`
- 组织 Claude + Codex 或单模型编码协作时：`服务端 Codex 协作规范.md`
- 判断子产品与模块归属时：`子产品与模块映射.md`

这些文档是 Picasso 项目专属规则，不是通用 skill 模板。
